# External Partners Portal

## Overview
AI-powered portal for suppliers, customers, and OEM partners with intelligent onboarding and assistance.

## Architecture
- Single multi-tenant portal with role-based experiences
- AI-first approach with Max assistant integration
- Shared database with main ERP system
- Unified authentication with portal-specific permissions

## User Types
1. **Suppliers**: PO management, delivery tracking, quality documents
2. **Customers**: Order placement, tracking, quality reports
3. **OEM/Retailers**: Strategic planning, demand forecasting, priority management

## AI Features
- Natural language setup and configuration
- Predictive alerts and recommendations
- Document intelligence and auto-extraction
- Personalized dashboards and workflows

## Deployment
Served from `/portal` route on main application
Access via: portal.yourdomain.com
